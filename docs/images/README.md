# Screenshots

Place screenshot images in this folder. The project README and docs reference
them by filename.

## Needed

| Filename | What to capture |
|----------|-----------------|
| `popup.png` | The extension popup showing both Pi-holes with green "Blocking" badges and the statistics lines. Referenced by `README.md`. |

## Optional (nice to have)

| Filename | What to capture |
|----------|-----------------|
| `popup-disabled.png` | The popup after clicking "Disable 5m", showing the orange "Disabled (5m)" badges. |
| `settings.png` | The popup with the Settings section expanded. |
| `about-debugging.png` | The Firefox `about:debugging` page with the temporary add-on loaded. |

## Tips

- Crop tightly to the popup window.
- Use a real (sanitized) setup — no real passwords or tokens visible.
- PNG format, keep file sizes small.
